# Tailwind CSS Task

## Project Setup

### Clone the repo

```sh
git clone git@github.com:sh4hids/task-tailwindcss.git
```

### Install packages

```sh
npm i
```

## Run the project

### Run in watch mode

```sh
npm run dev
```

### Build the project

```sh
npm run build
```

### Build preview

```sh
npm run preview
```
